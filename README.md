# DiscordX
A plugin to link minecraft servers with discord.

You can find the wiki [here](https://harrydev.gitbook.io/discordx).


# Features
- 2-way chat link

- Achievements

- Server info and whitelister

- Server started and stopped

- Join/Leave messages
